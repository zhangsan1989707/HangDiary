# 临时需求记录
## 主界面日记列表
- 日记列表支持设置卡片颜色
- 日记列表支持多选操作，如删除、设置卡片颜色、克隆、分享

## 左上角菜单抽屉
- Swipe Layout实现侧划菜单
- 抽屉菜单四项功能全部可用
### 设置
- 支持设置主题，如深、浅、系统主题

### 关于
- 项目使用的库
  - 数据库：Room
  - 网络请求：Retrofit
  - 图片加载：Glide
  - 日期时间处理：JodaTime
  - 日志：Timber
  - 权限请求：EasyPermissions
  - 其他：Kotlin、AndroidX、Material Design
